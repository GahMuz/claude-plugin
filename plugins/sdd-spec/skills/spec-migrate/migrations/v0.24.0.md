# v0.24.0 — Mise à jour de la section `models` dans config.json

Cette version réaffecte les modèles par défaut de deux agents et ajoute cinq nouvelles entrées dans `.sdd/config.json`.

## Step 1 : Mettre à jour les modèles existants

Dans `.sdd/config.json`, corriger les deux valeurs modifiées :

```bash
jq '.models.orchestrator = "sonnet" | .models["code-reviewer"] = "opus"' \
  .sdd/config.json > .sdd/config.tmp.json && mv .sdd/config.tmp.json .sdd/config.json
```

## Step 2 : Ajouter les nouveaux agents

Ajouter les cinq entrées manquantes si elles sont absentes :

```bash
jq '
  .models.planner //= "haiku" |
  .models["design-validator"] //= "sonnet" |
  .models["spec-reviewer"] //= "sonnet" |
  .models["doc-generator"] //= "haiku" |
  .models["doc-analyser"] //= "sonnet"
' .sdd/config.json > .sdd/config.tmp.json && mv .sdd/config.tmp.json .sdd/config.json
```

## Step 3 : Mettre à jour schemaVersion

Dans `.sdd/config.json`, remplacer la `schemaVersion` actuelle par `"0.24.0"`.
