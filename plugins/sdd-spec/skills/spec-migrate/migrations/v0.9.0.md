# v0.9.0

Aucune migration de données nécessaire.

Cette version introduit des améliorations fonctionnelles :
- `/spec-status` devient un skill autonome (remplace `/spec status`)
- `/spec recap` absorbe le skill `/continue`
- `spec-sync` devient l'agent `spec-reviewer` avec le skill thin wrapper `/spec-review`

Les fichiers `.sdd/` restent compatibles sans modification.

## Step 1 : Mettre à jour schemaVersion

Dans `.sdd/config.json`, remplacer `"schemaVersion": "0.8.0"` par `"schemaVersion": "0.9.0"`.
