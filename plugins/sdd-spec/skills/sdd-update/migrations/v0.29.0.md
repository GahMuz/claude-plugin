# v0.29.0 — Initialisation du répertoire `.sdd/decisions/`

Cette version introduit le support des ADRs (Architecture Decision Records). Un nouveau répertoire `.sdd/decisions/` et son registre doivent être créés dans les projets existants.

## Step 1 : Créer le répertoire `decisions`

```bash
mkdir -p .sdd/decisions
```

## Step 2 : Initialiser le registre des ADRs

```bash
cat > .sdd/decisions/registry.md << 'EOF'
# Registre des ADRs

| Numéro | Identifiant | Titre | Période | Statut |
|--------|-------------|-------|---------|--------|
EOF
```

## Step 3 : Mettre à jour schemaVersion

Dans `.sdd/config.json`, remplacer la `schemaVersion` actuelle par `"0.29.0"`.
```
