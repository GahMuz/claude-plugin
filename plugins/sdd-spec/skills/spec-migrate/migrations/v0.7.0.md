# v0.7.0 — Ajout du champ richRequirements

Le champ `richRequirements` (booléen) a été ajouté à `.sdd/config.json` lors de cette version.
Il est retiré en v0.8.0 car il n'influençait aucun comportement en pratique.

## Step 1 : Ajouter le champ si absent

Ouvrir `.sdd/config.json`. Si `richRequirements` est absent, ajouter :

```json
"richRequirements": false,
```

## Step 2 : Mettre à jour schemaVersion

Dans `.sdd/config.json`, remplacer `"schemaVersion": "0.6.0"` par `"schemaVersion": "0.7.0"`.
