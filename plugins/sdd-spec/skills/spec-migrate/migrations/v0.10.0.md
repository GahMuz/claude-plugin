# v0.10.0 — Spec actif unique, suppression de suspend

Cette version introduit la contrainte d'un seul spec actif par machine et supprime la commande `suspend`.

## Step 1 : Créer le répertoire local

```bash
mkdir -p .sdd/local
```

## Step 2 : Ajouter .sdd/local/ au .gitignore

Vérifier que `.gitignore` contient `.sdd/local/`. Si absent, ajouter la ligne.

## Step 3 : Corriger les specs suspendus

Pour chaque spec dans `.sdd/specs/registry.md` :
- Lire son `state.json`
- Si `currentPhase` vaut `"suspended"` : lire `suspendedFrom`, remplacer `currentPhase` par cette valeur, supprimer le champ `suspendedFrom`
- Mettre à jour `updatedAt`

## Step 4 : Mettre à jour schemaVersion

Dans `.sdd/config.json`, remplacer `"schemaVersion": "0.9.0"` par `"schemaVersion": "0.10.0"`.
