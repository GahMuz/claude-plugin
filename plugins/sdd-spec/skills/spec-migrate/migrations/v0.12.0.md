# v0.12.0 — Création du répertoire .sdd/local/

## Step 1 : Créer le répertoire local

Le hook `spec-sync-check.sh` lit `.sdd/local/active.json` depuis la v0.11.0, mais ce répertoire n'était pas créé pour les projets déjà initialisés.

```bash
mkdir -p .sdd/local
```

## Step 2 : Exclure .sdd/local/ du suivi git

Ajouter `.sdd/local/` dans `.gitignore` si absent :

```bash
grep -qxF '.sdd/local/' .gitignore || echo '.sdd/local/' >> .gitignore
```

## Step 3 : Mettre à jour schemaVersion

Dans `.sdd/config.json`, remplacer `"schemaVersion": "0.11.0"` par `"schemaVersion": "0.12.0"`.
